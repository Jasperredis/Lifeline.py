Lifeline.py v1.1 Beta 2.1
Lifeline.py v1.1 Beta 2.1 is now the only release of Lifeline.py available. It is a work-in-progress version of Lifeline.py v1.1 and brings some useful additions like:
- Linux installs
- Executable forms too!
- Revamped Updates tab
- Now fetches info in the intro
- Allows the info fetched to be refreshed
- Adds a ping on the title screen if you are not on the latest version
- Multiple pages can now be in the updates
- Descriptions can now be multiple lines long
- Revamped terminal output
- Now has colour
- Provides more feedback in processes
- Keep in mind that percentages are bugged out.
- Memory and resource management
- Lazy loading
- This is where the game only loads assets when it needs them and unloads them when it doesn't.
- The graphical settings menu cannot disable this yet. It must be done by modifying your save in json/saved.json.
- Automatic garbage collection every 300 frames
- Autosave
- Every 300 frames, the game will automatically update your save file.
- Just like lazy loading, this cannot be disabled through the graphical settings menu yet.
- Buttons to minimize/maximize and close the game
